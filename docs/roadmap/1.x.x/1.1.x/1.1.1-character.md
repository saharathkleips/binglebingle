# Milestone: Character

**Status:** Complete (2026-04-11)
**Requirements:** CHAR-01, CHAR-02

## Goal

The Character type and its two core functions — resolveCharacter() and isComplete() — implemented and fully tested.

## Requirements

- [x] **CHAR-01**: Character type `{ jamo: readonly string[] }` defined; resolveCharacter() reduces jamo list to combined jamo, syllable block, or null
- [x] **CHAR-02**: isComplete() returns true when resolveCharacter produces a syllable block in range U+AC00–U+D7A3

## Success Criteria

1. resolveCharacter({jamo:['ㅏ','ㅣ']}) returns 'ㅐ'; resolveCharacter({jamo:['ㅎ','ㅐ']}) returns '해'
2. resolveCharacter({jamo:['ㄱ','ㅏ','ㄱ']}) returns '각'; resolveCharacter({jamo:['ㄱ','ㅎ']}) returns null
3. isComplete({jamo:['ㅎ','ㅏ','ㄴ']}) returns true; isComplete({jamo:['ㄱ']}) returns false
4. All exported functions in src/lib/character/ have colocated Vitest tests that pass
