**Status:** Pending
**Affected Slices:** `src/lib/engine/`

## Goal

The three engine functions — `canSubmit()`, `evaluateGuess()`, and `calculateScore()` — enforce game rules and are independently unit-tested.

## Execution Ledger

_(Agent: update these checkboxes as tests pass and local commits are made)_

- [ ] **ENGN-01** (`src/lib/engine/`): `canSubmit()` validates that a submitted guess is constructible from the jamo pool via rotation and/or combination
- [ ] **ENGN-02** (`src/lib/engine/`): `evaluateGuess()` returns per-character CharacterResult (correct / present / absent) for a submitted guess against the target word
- [ ] **ENGN-03** (`src/lib/engine/`): `calculateScore()` returns a score value based on guess count

## Success Criteria

1. `canSubmit()` returns true for a guess constructible from the pool, false when a required jamo is not available
2. `evaluateGuess()` returns 'correct' for a character in the right position, 'present' for wrong position, 'absent' otherwise
3. `calculateScore()` returns a numeric score that decreases with more guesses
4. All exported functions have colocated Vitest tests that pass
