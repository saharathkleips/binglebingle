2. `evaluateGuess()` returns 'correct' for a character in the right position, 'present' for wrong position, 'absent' otherwise; empty slots always evaluate as 'absent'
3. `calculateScore()` returns `{ guessCount: guesses.length }`
4. All exported functions have colocated Vitest tests that pass
