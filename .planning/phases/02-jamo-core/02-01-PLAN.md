<objective>
Define all static Unicode data tables and derived runtime lookup maps that every jamo function depends on.

Purpose: This is the data foundation for the entire jamo domain. Rotation, combination, and syllable composition all look up values from these exports at runtime. If any codepoint is wrong or a table entry is missing, every downstream function silently fails — tests here catch that class of bug early.

Output: src/lib/jamo/jamo-data.ts (all exports), src/lib/jamo/jamo-data.test.ts (data invariant tests).
</objective>
