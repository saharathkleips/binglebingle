    All rules from CONTEXT.md `## combine() rules` section apply:

    - Empty target + incoming choseong → `{ choseong }`
    - Empty target + incoming jungseong → `{ jungseong }`
    - Choseong-only target + incoming choseong → combineJamo → `{ choseong: combined }` or null
    - Choseong-only target + incoming jungseong → `{ choseong, jungseong }`
    - Jungseong-only target + incoming jungseong → combineJamo → `{ jungseong: combined }` or null
    - Jungseong-only target + incoming choseong → `{ choseong: incoming, jungseong }`
    - Choseong+jungseong target + incoming jungseong → combineJamo on vowels → update jungseong, or null
    - Choseong+jungseong target + incoming choseong → `{ choseong, jungseong, jongseong: incoming.choseong }`

    - Full target + incoming jungseong → null (no 4-part syllables)
    - Full target + incoming jongseong → null
    - Any other combination not listed → null

    - No choseong AND no jungseong → null
    - Choseong only → return choseong string
    - Jungseong only → return jungseong string
    - Choseong + jungseong [+ optional jongseong] → composeSyllable(choseong, jungseong, jongseong)

    - Same codepoint range check U+AC00–U+D7A3 on resolveCharacter result
    - Requires at minimum choseong + jungseong (ㅇ not special-cased)

    - Choseong + jungseong → remove jungseong: `[{ choseong }]`
    - Choseong only → `[]` (nothing to decompose)
    - Empty → `[]`
